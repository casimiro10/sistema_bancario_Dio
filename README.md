💰 Sistema Bancário em Python

Projeto desenvolvido como parte do desafio da DIO (Digital Innovation One) com o objetivo de simular um sistema bancário simples utilizando Programação Orientada a Objetos em Python.

📌 Funcionalidades

O sistema permite:

👤 Criar cliente com:

Nome

CPF

Data de nascimento

Endereço

Email

Telefone

🏦 Criar conta corrente automaticamente

💵 Realizar depósitos

💸 Realizar saques

limite de valor por saque

limite de saques diários

📄 Exibir extrato bancário

histórico de transações

saldo atual

🧠 Conceitos aplicados

Este projeto utiliza conceitos importantes de programação:

Programação Orientada a Objetos (POO)

Classes e Herança

Classes Abstratas (ABC)

Encapsulamento

Organização de código

Registro de histórico de transações

🛠 Tecnologias utilizadas

Python 3

Biblioteca datetime

Biblioteca abc
